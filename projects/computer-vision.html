Computer Vision Research Project 2024

Real-Time Object Detection System

High-performance object detection and semantic segmentation pipeline using YOLOv8, optimized for real-time edge inference on resource-constrained hardware.

2024
PyTorch, YOLOv8, OpenCV, ONNX
View on GitHub

Overview

This project develops a complete computer vision pipeline for real-time object detection and instance segmentation, with a primary focus on edge deployment. The system is built on YOLOv8 and includes a complete optimization pipeline: training on custom datasets, knowledge distillation for model compression, ONNX export, and INT8 quantization for embedded inference.

The motivating application is assistive robotics — robots need to quickly and accurately perceive their environment on hardware without discrete GPUs. This system achieves real-time performance (>30 FPS) on NVIDIA Jetson Nano and 12 FPS on Raspberry Pi 5.

Key Features

👁

Multi-Task Detection

Simultaneous object detection, instance segmentation, and pose estimation in a single forward pass.

⚡

Edge Optimization

ONNX export with TensorRT and ONNX Runtime acceleration. INT8 quantization with calibration dataset.

📚

Custom Training

Fine-tuning pipeline with data augmentation, mosaic, and mix-up on custom annotated datasets.

📊

Evaluation Suite

Comprehensive benchmarking: mAP@50, mAP@50-95, FPS across hardware platforms, memory profiling.

Technical Pipeline

  1. Data Collection & Annotation — Custom dataset of 8,500 images annotated with Roboflow across 12 object categories relevant to robotics environments.
  2. Model Training — YOLOv8-m fine-tuned with cosine LR schedule, mosaic augmentation, and multi-scale training on a single A100 GPU (40GB).
  3. Knowledge Distillation — YOLOv8-m teacher → YOLOv8-n student distillation using response-based KD, recovering 94% of teacher accuracy at 4× smaller model size.
  4. Quantization — Post-training INT8 quantization using 200-image calibration set. 2.3× inference speedup with <1.5% mAP loss.
  5. Deployment — ONNX Runtime deployment with hardware-specific execution providers (CUDA, TensorRT, NNAPI, CoreML).

Tech Stack

Python 3.11 PyTorch 2.x Ultralytics YOLOv8 OpenCV ONNX Runtime TensorRT CUDA 12 Roboflow NumPy

Benchmark Results

Evaluated on custom robotics perception dataset (2,000 test images, 12 classes).
  • YOLOv8-m (FP32, GPU): mAP@50 = 87.2%, 94 FPS on A100
  • YOLOv8-n distilled (INT8, Jetson Nano): mAP@50 = 82.4%, 31 FPS
  • YOLOv8-n distilled (INT8, RPi 5): mAP@50 = 82.4%, 12 FPS
  • Model size reduction: 25.9 MB → 3.2 MB (8× smaller)
